# História do PHP

O PHP nasceu em 1994 quando Rasmus Lerdorf criou scripts em C para monitorar as visitas ao seu currículo online, lançando o pacote como PHP/FI em 1995 [1, 2]. A ferramenta não tinha a intenção de ser uma linguagem formal, servindo apenas como um utilitário para páginas estáticas [2].

A grande transformação ocorreu em 1997, quando os estudantes Andi Gutmans e Zeev Suraski encontraram limitações no sistema ao desenvolverem um e-commerce [3]. Eles cooperaram com Lerdorf para reescrever o núcleo do sistema, gerando o PHP 3 em 1998 [1]. Essa versão mudou o acrônimo para PHP: Hypertext Preprocessor e adicionou suporte a múltiplos bancos de dados [1].
