## Curiosidades

* **Código aberto:** O PHP é uma linguagem de código fonte aberto. Com isso, qualquer pessoa pode ajudar a melhorar o código fonte do PHP.
* **Popularidade:** O PHP é uma das linguagens mais usadas no mundo no desenvolvimento web.
* **Amplo suporte:** O PHP tem uma ampla biblioteca de funções embutidas, o que facilita a vida dos desenvolvedores com soluções prontas.
